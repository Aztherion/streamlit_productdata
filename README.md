# Product Metadata Management App

This is a Streamlit web application for managing product metadata, CRA planning, risk assessments, and support tracking.

## Features

- Product metadata entry, edit, and grouping
- CRA Plan management with EoL, Stop Sell in EU, or Compliance options
- VP approval tracking for Stop Sell plans
- Vulnerability handling questions
- Dashboard analytics and admin control
- Bulk import/export via CSV

## To Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment

1. Upload this folder to a GitHub repo
2. Go to [https://streamlit.io/cloud](https://streamlit.io/cloud)
3. Click "New App", select the repo and `app.py`
4. Deploy and use the public link!
